# 2D-RPG

A 2D RPG game project developed in Unity while learning game development concepts and systems.

## Features Implemented

* Player movement system
* Collider-based map interactions
* Basic battle system
* Login system
* Pause menu with logout functionality

## Current Status

The project is still in development.
The overworld system is currently being expanded and improved, while core movement and collision systems are functioning properly.

## Tech Used

* Unity
* C#
* SQLite
